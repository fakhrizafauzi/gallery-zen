# Zen Gallery | 禅ギャラリー 🉐

A premium, minimalist Japanese-inspired web archive viewer for images and documents. Designed with the aesthetics of **Wabi-Sabi** and **Modern Nihon** design.

![Screenshot](public/vite.svg)

## ✨ Features
- **Zen Aesthetic**: Light cream (Isago) backgrounds, Summ-e ink accents, and Washi paper textures.
- **Automatic Synchronization**: Artifacts dropped into local folders are automatically added to the web interface.
- **Artifact Intelligence**: View high-resolution previews and detailed metadata (file size, type, dates).
- **Universal Search**: Fast, "Shodou" inspired search bar to find artifacts instantly.
- **Ready for Cloud**: Fully optimized for GitHub Pages with automated deployment scripts.

## 📂 Project Structure
- `uploads/image/` — Drop your images here.
- `uploads/document/` — Drop your PDFs or text documents here.
- `src/` — React source code (Framer Motion + Tailwind).
- `scripts/` — Automatic file scanning and manifest generation.

## 🚀 Getting Started

### 1. Installation
```powershell
npm install
```

### 2. Development
Runs the app in development mode and watches for file changes.
```powershell
npm run dev
```

### 3. Build & Deploy
Bundles the app for production and pushes to GitHub Pages.
```powershell
npm run deploy
```

## 🛠️ Technology Stack
- **Framework**: React 19 + Vite 8
- **Styling**: Tailwind CSS v3 (Stable Static)
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Typography**: Noto Serif JP & Noto Sans JP
